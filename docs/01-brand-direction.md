# Precision Gear — Brand Direction v1

## Visual reference baseline (industry)
- gleason.com — strong industrial premium tone, clear value proposition, engineering-first messaging.
- khkgears.us — product clarity, practical CTA flow.
- rushgears.com — speed/value blocks and urgent quoting emphasis.

## Color system (proposed)
- Primary (Deep Steel): `#0F172A`
- Secondary (Machine Blue): `#1D4ED8`
- Accent (Signal Cyan): `#06B6D4`
- Neutral 900: `#111827`
- Neutral 600: `#4B5563`
- Neutral 100: `#F3F4F6`
- Surface: `#FFFFFF`
- Success CTA: `#16A34A`

## Why this palette
- Looks technical and trustworthy (B2B/engineering feel).
- Keeps contrast high for readability.
- Blue-cyan accents fit precision/mechanics positioning.

## Typography
- Headings: Inter / Manrope (700)
- Body: Inter (400/500)
- Numbers/spec tables: JetBrains Mono (optional)

## UI style
- Clean grid, medium spacing, large technical photos.
- Section cards with specs and tolerances.
- Sticky CTA: “Надіслати креслення”.

## CTA language
- Primary: "Отримати прорахунок"
- Secondary: "Надіслати креслення"
- Tertiary: "Консультація технолога"
