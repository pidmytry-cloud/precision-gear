## SEO clusters
### Commercial
- виготовлення шестерень на замовлення
- зубонарізання шестерень
- зубошліфування шестерень
- черв’ячні передачі виготовлення
- механічна обробка деталей за кресленням

### Informational (blog)
- як вибрати матеріал для шестерні
- допуски та точність зубчастих коліс
- модуль, кут тиску, тип зуба — пояснення
- коли потрібне зубошліфування
- помилки у проектуванні передач

## Internal linking rules
- Кожна сторінка послуги -> 2 релевантні кейси + 2 статті блогу + RFQ.
- Кожен кейс -> послуги + галузь + контакт.
- Блог -> послуги (контекстні блоки "потрібне виготовлення?").
